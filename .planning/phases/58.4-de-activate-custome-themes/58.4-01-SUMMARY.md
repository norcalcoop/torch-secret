## Accomplishments

- `export const RETRO_ENABLED = false` in `theme-toggle.ts` — the single flip-point to restore retro themes post-launch
- `createThemeDropdown()` panel now shows only Light/Dark/System; no separator, no "Retro Pro" label, no pixel grid
- `app.ts` guards `initRetroThemeListener()` and `applyRetroTheme()` behind the gate; `else` branch clears `retro-theme` from localStorage
- FOWT inline script in `index.html` gates the R-object behind `var RETRO_ENABLED = false` and `else { localStorage.removeItem('retro-theme') }` — prevents any retro flicker on hard refresh for users with a stale key
