Log in to app.posthog.com and create the following in the Torch Secret project:

**A. Launch Dashboard**

Create a new Dashboard named "Torch Secret Launch".

Add these 5 widgets (all as Trends chart type):

1. **DAU (Daily Active Users)**
   - Event: `$pageview`
   - Aggregation: Unique users
   - Granularity: Daily
   - Widget title: "DAU"

2. **Secrets created / day**
   - Event: `secret_created`
   - Aggregation: Event count
   - Granularity: Daily
   - Widget title: "Secrets Created / Day"

3. **Registrations / day**
   - Event: `user_registered`
   - Aggregation: Event count
   - Granularity: Daily
   - Widget title: "Registrations / Day"

4. **Pro upgrades / day**
   - Event: `subscription_activated`
   - Aggregation: Event count
   - Granularity: Daily
   - Widget title: "Pro Upgrades / Day"

5. **Top pages (route breakdown)**
   - Event: `$pageview`
   - Aggregation: Event count
   - Breakdown by: `$pathname`
   - Filter: `$pathname` is one of `/`, `/create`, `/pricing`, `/dashboard`
   - Widget title: "Top Pages by Route"

---

**B. Conversion Funnels**

Navigate to Product Analytics → Funnels. Create these two:

1. **Primary Conversion Funnel**
   - Name: "Free-to-Paid Conversion"
   - Steps (in order):
     1. `$pageview` — filter: `$pathname = /`
     2. `secret_created`
     3. `user_registered`
     4. `subscription_activated`

2. **Conversion Prompt Funnel**
   - Name: "Conversion Prompt Effectiveness"
   - Steps (in order):
     1. `conversion_prompt_shown`
     2. `conversion_prompt_clicked`
     3. `user_registered`

---

**C. Cohorts**

Navigate to Data Management → Cohorts (or People → Cohorts). Create these three:

1. **Pro Users**
   - Name: "Pro Users"
   - Filter: Person property `tier` equals `pro`

2. **Free Registered Users**
   - Name: "Free Registered Users"
   - Filter: Person property `tier` equals `free` AND person has performed event `user_registered` at least once

3. **Power Users (Dashboard)**
   - Name: "Power Users (Dashboard)"
   - Filter: Person has performed event `dashboard_viewed` in the last 30 days, at least 1 time
