<task type="checkpoint:human-action">
  <name>Task 2: Retrieve Resend DKIM value and add all 9 DNS records in Cloudflare</name>
  <action>
    Open Cloudflare → Websites → torchsecret.com → DNS → Records.

    **Step A: Get Resend DKIM value**
    1. In a separate tab, open Resend dashboard → Domains
    2. If torchsecret.com is not yet listed, click "Add Domain" and add it
    3. On the torchsecret.com domain detail page, copy the DKIM TXT value (the full "p=..." public key string)

    **Step B: Add Resend records (3 records)**

    Record 1 — Resend DKIM:
    - Type: TXT
    - Name: `resend._domainkey` (NOT resend._domainkey.torchsecret.com — Cloudflare appends domain automatically)
    - Value: paste the full DKIM TXT value from Resend dashboard
    - Proxy: DNS Only (grey) — TXT records cannot be proxied; verify no orange cloud appears

    Record 2 — Resend SPF:
    - Type: TXT
    - Name: `send`
    - Value: `v=spf1 include:amazonses.com ~all`
    - Proxy: DNS Only (cannot be proxied)

    Record 3 — Resend MX:
    - Type: MX
    - Name: `send`
    - Mail server: `feedback-smtp.us-east-1.amazonses.com`
    - Priority: 10
    - Proxy: DNS Only (MX records cannot be proxied)

    **Step C: Add Loops.so records (5 records)**

    Records 4, 5, 6 — Loops DKIM CNAMEs (use exact values from Task 1):
    For EACH of the 3 DKIM selectors:
    - Type: CNAME
    - Name: [selector prefix only, e.g., `loops1._domainkey` — NOT the full FQDN]
    - Target: [full amazonaws.com target from Loops dashboard]
    - Proxy: MUST be DNS Only (grey cloud) — click the orange cloud icon to toggle it grey BEFORE saving
    - WARNING: Cloudflare defaults new CNAMEs to proxied. Proxied DKIM CNAMEs break verification permanently.

    Record 7 — Loops SPF:
    - Type: TXT
    - Name: `envelope`
    - Value: `v=spf1 include:amazonses.com ~all`
    - Proxy: DNS Only

    Record 8 — Loops MX:
    - Type: MX
    - Name: `envelope`
    - Mail server: [exact value from Loops dashboard — e.g., feedback-smtp.us-east-1.amazonses.com]
    - Priority: 10
    - Proxy: DNS Only

    **Step D: Add DMARC record (1 record)**

    Record 9 — DMARC:
    - Navigate to DNS → Records (NOT Email → DMARC Management wizard — using the wizard adds a Cloudflare rua= address)
    - Type: TXT
    - Name: `_dmarc`
    - Value: `v=DMARC1; p=none; rua=mailto:admin@torchsecret.com`
    - Proxy: DNS Only

    **Step E: Verify apex @ SPF is untouched**
    - Find the existing @ TXT record — it should show `v=spf1 include:_spf.mx.cloudflare.net ~all`
    - Confirm no amazonses.com appears in the apex @ SPF record
    - DO NOT modify it

    **Step F: Confirm record count**
    The DNS list should now contain 9 new records compared to before this task. Verify all 3 Loops DKIM CNAMEs show the grey cloud icon (DNS Only).
