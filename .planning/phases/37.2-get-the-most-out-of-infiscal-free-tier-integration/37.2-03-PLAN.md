<task type="checkpoint:human-action">
  <name>Task 1: Set GitHub Actions repository variable + configure Render Secret Sync</name>
  <files>none (no files modified — human action configures external platforms)</files>
  <action>
    This is a human-only step. Complete both platform wiring actions in GitHub and Infisical/Render dashboards.

    **Step 1 — Set GitHub Actions repository variable**
    This is a variable (non-sensitive), not a secret — use the vars context in GitHub, not secrets.

    1. Go to: https://github.com/norcalcoop/secureshare/settings/variables/actions
    2. Click "New repository variable"
    3. Name: `INFISICAL_IDENTITY_ID`
    4. Value: (paste your INFISICAL_IDENTITY_ID UUID from Plan 01 / 37.2-01-SUMMARY.md)
    5. Click "Add variable"

    **Step 2 — Configure Infisical Render Secret Sync**
    This syncs `prod` environment secrets from Infisical to Render automatically.

    2a. Get a Render API key:
    - Go to: https://dashboard.render.com/u/settings (Account Settings)
    - Under "API Keys", click "Create API Key"
    - Name it: `infisical-sync`
    - Copy the key value

    2b. Add Render connection in Infisical:
    - Go to: https://app.infisical.com
    - Open the `torch-secret` project
    - Go to: Project > Integrations > App Connections
    - Click "Add Connection" > Render
    - Paste the Render API key
    - Save

    2c. Create Secret Sync:
    - In Infisical: Project > Integrations > Secret Syncs > New Sync
    - Choose: Render
    - Source: Environment = `prod`, Secret Path = `/`
    - Destination: Select your Render service (not Environment Group — select the specific web service)
    - Enable: Auto-Sync
    - Click "Create Sync"
    - Trigger initial sync: click "Sync Now" to push all prod secrets to Render immediately

    2d. Verify in Render dashboard:
    - Go to your Render service > Environment
    - Confirm the secrets (BETTER_AUTH_SECRET, STRIPE_SECRET_KEY, etc.) appear in the environment variables list
    - Note: Render may trigger a redeploy when env vars change — this is expected behavior
  </action>
  <verify>
    - GitHub: https://github.com/norcalcoop/secureshare/settings/variables/actions shows INFISICAL_IDENTITY_ID set to a UUID
    - Infisical Project > Integrations page shows Render sync with green/active status
    - Render service Environment tab shows secrets (BETTER_AUTH_SECRET, STRIPE_SECRET_KEY, etc.) present
  </verify>
  <done>
    GitHub repository variable INFISICAL_IDENTITY_ID is set. Infisical Render Secret Sync shows green status. Render service environment has secrets synced from Infisical prod environment.
  </done>
