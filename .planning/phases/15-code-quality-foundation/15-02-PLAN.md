   ```

2. Line 48 — `paddedBytes` from `padPlaintext()` returns `Uint8Array` (inferred as `Uint8Array<ArrayBufferLike>`). The `crypto.subtle.encrypt` third parameter requires `BufferSource`. Fix by asserting at the call site:

   ```
   So the full encrypt call becomes:
   ```typescript
   const ciphertextBuffer = await crypto.subtle.encrypt(
     { name: ALGORITHM, iv },
     key,
     paddedBytes as Uint8Array<ArrayBuffer>,
   );
   ```

**keys.ts** — 1 error (Uint8Array<ArrayBufferLike> not assignable to BufferSource):

   const rawKey = base64UrlToUint8Array(base64url);
   return crypto.subtle.importKey(
     'raw',
     rawKey as Uint8Array<ArrayBuffer>,
     ALGORITHM,
     false,
     ['decrypt'],
   );
   ```

**rate-limit.ts** — 1 error (spread argument tuple type):

   ```
