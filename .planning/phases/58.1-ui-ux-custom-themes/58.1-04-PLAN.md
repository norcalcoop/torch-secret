```typescript
import type { RetroTheme } from './retro-data.js';

const EFFECTS_ROOT_ID = 'retro-effects-root';
let currentCleanup: (() => void) | null = null;

export function unmountRetroEffects(): void {
  if (currentCleanup) {
    currentCleanup();
    currentCleanup = null;
  }
  document.getElementById(EFFECTS_ROOT_ID)?.remove();
}

export function mountRetroEffects(id: string, theme: RetroTheme): void {
  unmountRetroEffects(); // always clean up before mounting new

  const root = document.createElement('div');
  root.id = EFFECTS_ROOT_ID;
  root.style.cssText = 'position:fixed;inset:0;pointer-events:none;z-index:0;overflow:hidden;';
  document.body.appendChild(root);

  const cleanups: Array<() => void> = [];

  if (theme.matrixMode) cleanups.push(mountMatrixRain(root));
  if (theme.pongMode) cleanups.push(mountPongBall(root));
  if (theme.c64Mode) cleanups.push(mountBlink(root, '#a0a0ff'));
  if (theme.dosMode) {
    cleanups.push(mountDosTyper(root));
    cleanups.push(mountBlink(root, '#ffffff'));
  }
  if (theme.scanlines) cleanups.push(mountScanlineOverlay(root));
  if (theme.floats.length > 0) cleanups.push(mountFloatingEmojis(root, theme));

  currentCleanup = () => { for (const fn of cleanups) fn(); };
}
```

Implement each mount function:

**mountMatrixRain(root):** Read RESEARCH.md Pattern 5 MatrixRain spec. Pure CSS animation — no setInterval. Check `matchMedia('(prefers-reduced-motion: reduce)').matches` at start; if true, return `() => {}` immediately. Create 18 column divs with `writing-mode: vertical-rl`. Use CSS animation referencing `@keyframes mRain` (added to styles.css). Returns cleanup that removes created elements.

**mountPongBall(root):** Read RESEARCH.md Pattern 5 PongBall spec. setInterval at 16ms. Check prefers-reduced-motion first. Create two paddle divs, one ball div, one center line. Bounce logic per spec. Returns cleanup: `() => clearInterval(intervalId); root.querySelectorAll('.pong-*')?.forEach(e => e.remove())` (or remove from root directly).

**mountDosTyper(root):** Read RESEARCH.md Pattern 5 DosTyper spec. 13-line setTimeout chain. Check prefers-reduced-motion: if reduced, render all 13 lines at once immediately. Returns cleanup that clears any pending timeouts via a collected array.

**mountBlink(root, color):** Read RESEARCH.md Pattern 5 Blink spec. setInterval 500ms; toggle visibility. Check prefers-reduced-motion: if reduced, render static visible char. Returns cleanup: `() => clearInterval(intervalId)`.

**mountFloatingEmojis(root, theme):** Read RESEARCH.md Pattern 5 FloatingEmojis spec. Check prefers-reduced-motion first; return no-op if reduced. Create 5 fixed-position emoji divs with `@keyframes floatY` animation. Returns cleanup that removes elements.

**mountScanlineOverlay(root):** CSS-only, no animation. Create a fixed-position div with `repeating-linear-gradient` background per spec. No prefers-reduced-motion guard (texture is not motion). Returns cleanup.
