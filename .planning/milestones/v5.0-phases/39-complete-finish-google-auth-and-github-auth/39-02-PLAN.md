**Better Auth callback URL pattern:**
Better Auth auto-generates the OAuth callback at `{BETTER_AUTH_URL}/api/auth/callback/github`.
- Dev: `BETTER_AUTH_URL=http://localhost:3000` → callback is `http://localhost:3000/api/auth/callback/github`
- Prod: `BETTER_AUTH_URL=https://torchsecret.com` → callback is `https://torchsecret.com/api/auth/callback/github`

**GitHub OAuth Apps vs Google OAuth:**
GitHub does NOT support multiple callback URLs per OAuth App. The "Authorization callback URL" field accepts a single URL. Two separate apps are required: one for dev, one for prod.

**GitHub scope:**
Better Auth's GitHub provider requests `user:email` scope by default. Do NOT set `disableDefaultScope: true`. Without `user:email`, GitHub returns a null email for users with private email settings, causing sign-up to fail silently (redirect to `/login?error=oauth`).
