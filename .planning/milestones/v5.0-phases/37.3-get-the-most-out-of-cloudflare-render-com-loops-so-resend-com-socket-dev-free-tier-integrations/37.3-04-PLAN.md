**1. Add a `deleteAccount()` async function** (near the bottom of dashboard.ts, before the main render function):

```typescript
async function deleteAccount(userEmail: string): Promise<void> {
  try {
    const res = await fetch('/api/me', { method: 'DELETE' });
    if (res.ok) {
      showToast('Your account has been deleted', 'success');
      navigate('/');
    } else {
      showToast('Failed to delete account. Please try again.', 'error');
    }
  } catch {
    showToast('Failed to delete account. Please try again.', 'error');
  }
}
```

  - A "Cancel" button
- Keyboard: Escape key closes the modal without deleting
