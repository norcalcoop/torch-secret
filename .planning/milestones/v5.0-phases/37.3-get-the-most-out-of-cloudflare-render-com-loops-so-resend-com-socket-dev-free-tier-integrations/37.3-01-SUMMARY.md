- `BETTER_AUTH_TRUSTED_ORIGINS` — CSRF protection trusted origins

- `RESEND_AUDIENCE_ID` — Phase 36 email capture audience sync

PostHog REST API keys (`POSTHOG_PERSONAL_API_KEY`, `POSTHOG_PROJECT_ID`) added as commented-out stubs with an explanatory note that they are not runtime requirements.

**Task 2 — CI auto-triggers:**

Replaced the temporary `workflow_dispatch`-only `on:` block with:

```yaml

  push:
    branches: [main]
  pull_request:
  workflow_dispatch:
```

This activates `autoDeployTrigger: checksPass` in `render.yaml` — production deploys now only occur after all CI checks pass on the main branch.
