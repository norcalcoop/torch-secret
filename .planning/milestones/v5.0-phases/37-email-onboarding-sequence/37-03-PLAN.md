  <files>N/A — Loops UI configuration, no code files modified</files>
  <action>
The app fires `eventName: 'registered'` with contact properties `{ firstName, marketingConsent, subscriptionTier }` after every new registration. You must now build the loop in the Loops dashboard that responds to this event. There is no API for creating loops — this must be done manually.

**Steps:**

1. Log in to your Loops dashboard at https://app.loops.so

2. Navigate to **Loops** (sidebar) and click **New Loop**.

3. Set the **Trigger** to: **Event received** → event name: `registered`

4. Build the loop with these 7 nodes in order:

   **Node 1 — Welcome email (no audience filter; sends to everyone):**
   - Action: Send email
   - Subject: `Hey {{firstName}}, welcome to Torch Secret 👋`
   - Body: see email copy below
   - No audience filter on this step

   **Node 2 — Timer: 3 days**
   - Delay: 3 days after the previous node

   **Node 3 — Audience filter:**
   - Condition: `marketingConsent` equals `true`

   **Node 4 — Day-3 key features email (only reaches consent=true users):**
   - Action: Send email
   - Subject: `{{firstName}}, here's what Torch Secret can do`
   - Body: see email copy below

   **Node 5 — Timer: 4 more days** (total = 7 days from registration)

   **Node 6 — Audience filter:**
   - Condition: `marketingConsent` equals `true` AND `subscriptionTier` does not equal `pro`

   **Node 7 — Day-7 upgrade prompt email:**
   - Action: Send email
   - Subject: `{{firstName}}, unlock the full Torch Secret experience`
   - Body: see email copy below

5. Set the **From** name and email to match your Resend sender (e.g., `Torch Secret <hello@torchsecret.com>`)

6. **Publish the loop** (set to Active).
