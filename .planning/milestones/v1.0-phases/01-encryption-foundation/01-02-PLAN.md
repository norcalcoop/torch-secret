**GREEN phase -- implement to pass.**

Create `client/src/crypto/padding.ts` exporting three functions:

- `padmeLength(len: number): number` -- the PADME algorithm:
  1. If `len <= MIN_PADDED_SIZE` (256), return `MIN_PADDED_SIZE`
  2. `const e = Math.floor(Math.log2(len))`
  3. `const s = Math.floor(Math.log2(e)) + 1`
  4. `const lastBits = e - s`
  5. `const bitMask = (1 << lastBits) - 1`
  6. `return (len + bitMask) & ~bitMask`

- `padPlaintext(data: Uint8Array): Uint8Array` -- padding format is `[4-byte uint32 BE original length] + [original data] + [zero fill to padmeLength]`:
  1. `const totalDataLen = 4 + data.length` (4 bytes for length prefix)
  2. `const paddedLen = padmeLength(totalDataLen)`
  3. Create `new Uint8Array(paddedLen)` (zero-filled by default)
  4. Write `data.length` as big-endian uint32 in first 4 bytes using `DataView.setUint32(0, data.length, false)`
  5. Copy `data` starting at offset 4
  6. Return the padded array

- `unpadPlaintext(padded: Uint8Array): Uint8Array` -- extract original data:
  1. Read original length from first 4 bytes using `DataView.getUint32(0, false)`
  2. Validate: if `originalLen > padded.length - 4`, throw `new Error('Invalid padding: length exceeds padded data')`
  3. Return `padded.slice(4, 4 + originalLen)`

Import `MIN_PADDED_SIZE` from `./constants.ts`. Do NOT import any third-party libraries.

Run tests -- ALL MUST PASS.
