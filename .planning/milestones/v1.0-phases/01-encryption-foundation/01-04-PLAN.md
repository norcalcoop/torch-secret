Create `client/src/crypto/__tests__/encrypt.test.ts` with tests covering:

1. Return shape:
   - `encrypt('hello')` returns an object matching `EncryptResult` shape: `{ payload: { ciphertext: string }, key: CryptoKey, keyBase64Url: string }`
   - `payload.ciphertext` is a non-empty string (base64 encoded)
   - `keyBase64Url` is 43 characters, URL-safe (`/^[A-Za-z0-9_-]+$/`)

2. IV prepending:
   - Decode `payload.ciphertext` from base64 -> the first 12 bytes are the IV, remaining bytes are ciphertext+tag
   - The total decoded length is >= IV_LENGTH (12) + TAG_LENGTH (16) + MIN_PADDED_SIZE (256) = 284 bytes for any input (because padding minimum is 256)

3. Uniqueness:
   - Two calls to `encrypt('same text')` produce different `payload.ciphertext` (different IVs)
   - Two calls to `encrypt('same text')` produce different `keyBase64Url` (different keys)

4. Padding verification:
   - `encrypt('hi')` and `encrypt('a'.repeat(200))` produce `payload.ciphertext` of the SAME base64 length (both fall in 256-byte minimum padding tier, so ciphertext is 256 + 12 IV + 16 tag = 284 bytes -> same base64 length)
   - `encrypt('a'.repeat(300))` produces a LONGER ciphertext than `encrypt('hi')` (different padding tier)

5. Empty string:
   - `encrypt('')` succeeds and produces valid output (edge case: 0-length plaintext pads to 256 bytes)

6. Large input:
   - `encrypt('x'.repeat(10000))` succeeds (max secret size)

7. Unicode:
   - `encrypt('Hello')` succeeds (emoji/unicode chars)

**GREEN phase -- implement to pass.**

Create `client/src/crypto/encrypt.ts` exporting one function:

- `encrypt(plaintext: string): Promise<EncryptResult>`:
  1. `const encoder = new TextEncoder()`
  2. `const plaintextBytes = encoder.encode(plaintext)` -- UTF-8 encoding
  3. `const paddedBytes = padPlaintext(plaintextBytes)` -- PADME padding
  4. `const { key, keyBase64Url } = await generateKey()` -- fresh 256-bit key
  5. `const iv = crypto.getRandomValues(new Uint8Array(IV_LENGTH))` -- fresh 96-bit IV
  6. `const ciphertextBuffer = await crypto.subtle.encrypt({ name: ALGORITHM, iv }, key, paddedBytes)` -- AES-GCM encrypt
  7. Prepend IV: `const combined = new Uint8Array(iv.length + ciphertextBuffer.byteLength); combined.set(iv, 0); combined.set(new Uint8Array(ciphertextBuffer), iv.length)`
  8. `const ciphertext = uint8ArrayToBase64(combined)` -- base64 encode for transport
  9. `return { payload: { ciphertext }, key, keyBase64Url }`

Import from: `./keys` (generateKey), `./padding` (padPlaintext), `./encoding` (uint8ArrayToBase64), `./constants` (ALGORITHM, IV_LENGTH), `./types` (EncryptResult, EncryptedPayload).

Do NOT use `Math.random`. Do NOT import third-party libraries. IV MUST come from `crypto.getRandomValues`.

   - Test with short string: `'hi'`
   - Test with medium string: `'a'.repeat(500)`
   - Test with max-length string: `'x'.repeat(10000)`
   - Test with empty string: `''`
   - Test with Unicode: `'Hello from the future'`

   - Test with multi-byte UTF-8: `'cafe\u0301'` (combining accent)

2. Wrong key rejection:
   - Encrypt with key A, try to decrypt with key B -> must throw (OperationError from crypto.subtle.decrypt)
   - The error should be caught and NOT expose internals (just throw a clear error)

3. Tampered ciphertext:
   - Encrypt a secret, decode ciphertext from base64, flip one bit in the middle of the ciphertext portion (not the IV), re-encode to base64, try to decrypt -> must throw
   - This validates AES-GCM's authentication tag catches tampering

4. Tampered IV:
   - Encrypt a secret, decode ciphertext, flip one bit in the IV portion (first 12 bytes), re-encode, try to decrypt -> must throw

5. Truncated ciphertext:
   - Pass a base64 string that decodes to fewer than 12 + 16 = 28 bytes -> must throw (not enough data for IV + tag)

Run tests -- ALL MUST FAIL.

**GREEN phase -- implement to pass.**

Create `client/src/crypto/decrypt.ts` exporting one function:

- `decrypt(ciphertextBase64: string, keyBase64Url: string): Promise<string>`:
  1. `const combined = base64ToUint8Array(ciphertextBase64)` -- decode from base64
  2. Validate: `if (combined.length < IV_LENGTH + TAG_LENGTH) throw new Error('Invalid ciphertext: too short')`
  3. `const iv = combined.slice(0, IV_LENGTH)` -- extract IV (first 12 bytes)
  4. `const ciphertextWithTag = combined.slice(IV_LENGTH)` -- rest is ciphertext+tag
  5. `const key = await importKeyFromBase64Url(keyBase64Url)` -- import key
  6. Decrypt in try/catch: `const paddedBuffer = await crypto.subtle.decrypt({ name: ALGORITHM, iv }, key, ciphertextWithTag)` -- if this throws (wrong key, tampered data), catch and throw a generic error: `new Error('Decryption failed: invalid key or corrupted data')`
  7. `const paddedBytes = new Uint8Array(paddedBuffer)`
  8. `const plaintextBytes = unpadPlaintext(paddedBytes)` -- remove PADME padding
  9. `const decoder = new TextDecoder()`
  10. `return decoder.decode(plaintextBytes)`

Import from: `./keys` (importKeyFromBase64Url), `./padding` (unpadPlaintext), `./encoding` (base64ToUint8Array), `./constants` (ALGORITHM, IV_LENGTH, TAG_LENGTH).
