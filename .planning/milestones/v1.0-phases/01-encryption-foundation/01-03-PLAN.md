Create `client/src/crypto/__tests__/keys.test.ts` with tests covering:

   - Returns an object with `key` (CryptoKey) and `keyBase64Url` (string)

   - `key.algorithm.name` is `'AES-GCM'`

   - `keyBase64Url` contains only URL-safe characters: matches `/^[A-Za-z0-9_-]+$/`

2. Key uniqueness tests:
   - Two calls to `generateKey()` produce different `keyBase64Url` values
   - 10 calls to `generateKey()` produce 10 unique `keyBase64Url` values (Set size === 10)

3. `exportKeyToBase64Url(key)` tests:
   - Given a CryptoKey from `generateKey()`, returns the same base64url string as `generateKey()` returned
   - Output is 43 characters, URL-safe

4. `importKeyFromBase64Url(base64url)` tests:
   - Given a base64url string from `exportKeyToBase64Url`, returns a CryptoKey
   - Imported key has `extractable === false` (security: no need to re-export after import)
   - Imported key has `usages` containing `'decrypt'` (decrypt-only for the receiving side)
   - Imported key `algorithm.name` is `'AES-GCM'`

5. Export/import round-trip tests:
   - Generate key -> export to base64url -> import from base64url -> use imported key to decrypt something encrypted with the original key (functional round-trip)
   - This requires a minimal encrypt/decrypt inline in the test using `crypto.subtle.encrypt`/`decrypt` directly (not importing from encrypt.ts which does not exist yet)

**GREEN phase -- implement to pass.**

Create `client/src/crypto/keys.ts` exporting three functions:

- `generateKey(): Promise<{ key: CryptoKey; keyBase64Url: string }>`:
  1. `const key = await crypto.subtle.generateKey({ name: ALGORITHM, length: KEY_LENGTH }, true, ['encrypt', 'decrypt'])`
  2. Export key to base64url using `exportKeyToBase64Url(key)`
  3. Return `{ key, keyBase64Url }`

- `exportKeyToBase64Url(key: CryptoKey): Promise<string>`:
  1. `const rawKey = await crypto.subtle.exportKey('raw', key)`
  2. `return uint8ArrayToBase64Url(new Uint8Array(rawKey))`

- `importKeyFromBase64Url(base64url: string): Promise<CryptoKey>`:
  1. `const rawKey = base64UrlToUint8Array(base64url)`
  2. `return crypto.subtle.importKey('raw', rawKey, ALGORITHM, false, ['decrypt'])`
  Note: `extractable` is `false` (no need to re-export) and usages is `['decrypt']` only (receiving side only decrypts).

Import `ALGORITHM`, `KEY_LENGTH` from `./constants.ts`. Import `uint8ArrayToBase64Url`, `base64UrlToUint8Array` from `./encoding.ts`.

Do NOT use `Math.random` anywhere. Do NOT import any third-party crypto library.

Run tests -- ALL MUST PASS.
