  <action>
  **index.html changes:**

  Add three elements inside `<body>`, before the existing `<div id="app">`:

  1. **Skip link** as the first child of `<body>`:
     ```html
     <a href="#main-content" class="sr-only focus:not-sr-only focus:fixed focus:top-4 focus:left-4 focus:z-50 focus:bg-primary-600 focus:text-white focus:px-4 focus:py-2 focus:rounded-lg focus:ring-2 focus:ring-primary-500 focus:ring-offset-2">
       Skip to main content
     </a>
     ```

  2. **Route announcer** (visually hidden live region, starts empty):
     ```html
     <div id="route-announcer" class="sr-only" aria-live="polite" aria-atomic="true"></div>
     ```

  3. **Wrap `<div id="app">` in `<main id="main-content">`:**
     Replace:
     ```html
     <div id="app" class="max-w-2xl mx-auto px-4 py-8"></div>
     ```
     With:
     ```html
     <main id="main-content">
       <div id="app" class="max-w-2xl mx-auto px-4 py-8"></div>
     </main>
     ```

  **router.ts changes:**

  1. Add a `updatePageMeta(title: string)` function that:
     - Sets `document.title` to `${title} - SecureShare`
     - Clears `#route-announcer` textContent, then uses `requestAnimationFrame` to set it to `title` (the clear-then-set pattern triggers announcement even for repeated titles)

  2. Add a `focusPageHeading()` function that:
     - Queries `document.querySelector('h1')`
     - Sets `tabindex="-1"` on it (allows programmatic focus without adding to tab order)
     - Calls `.focus()` on it

  3. Update `handleRoute()` to call `updatePageMeta()` BEFORE the dynamic import for each route:
     - `/` route: `updatePageMeta('Share a Secret')`
     - `/secret/` route: `updatePageMeta("You've Received a Secret")`
     - Default (not found): `updatePageMeta('Page Not Found')`
     Then chain `.then(() => focusPageHeading())` after each page render `.then()` call.

  4. The confirmation page is rendered via `renderConfirmationPage()` directly from create.ts (state-based, not URL-based), so it does NOT go through `handleRoute()`. The confirmation page itself will call `updatePageMeta` -- export it from router.ts so confirmation.ts can import it. Add it to the exports: `export { navigate, initRouter, updatePageMeta, focusPageHeading }`.

  5. Update `showLoadError()` to also set `document.title = 'Error - SecureShare'`.
  </action>
