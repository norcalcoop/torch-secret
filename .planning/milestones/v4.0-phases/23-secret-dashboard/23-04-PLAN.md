```typescript
function formatDate(dateStr: string): string {
  return new Intl.DateTimeFormat(undefined, {
    month: 'short', day: 'numeric', year: 'numeric',
    hour: '2-digit', minute: '2-digit',
  }).format(new Date(dateStr));
}
```

**Table rendering:**

Use semantic HTML: `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th scope="col">`, `<td>`. Add `aria-label="Your secrets"` to the `<table>` element.

Table wrapper: horizontally scrollable container (`overflow-x-auto`) so the table is usable on mobile. Add `min-w-full` to the `<table>`.

Columns: Label, Created, Expires, Status, Notification, Delete.

For each row (`DashboardSecretItem`):
- **Label**: display `item.label ?? '—'` (em-dash for no label). `text-text-primary text-sm`.
- **Created**: `formatDate(item.createdAt)`. `text-text-secondary text-sm whitespace-nowrap`.
- **Expires**: `formatDate(item.expiresAt)`. `text-text-secondary text-sm whitespace-nowrap`.
- **Status**: `createStatusBadge(item.status)`.
- **Notification**: `item.notify ? 'On' : '—'`. Use `Bell` Lucide icon with text "On" for notify=true; em-dash for false. `text-text-muted text-sm`.
- **Delete button**: `<button type="button">` with `Trash2` icon. Always rendered. Active status: full opacity, pointer cursor, `hover:text-danger focus:ring-2 focus:ring-accent`. Non-active: `opacity-40 cursor-not-allowed`, `disabled` attribute set, `aria-disabled="true"`. All delete buttons get `aria-label="Delete secret"` (or include label name if present: `"Delete ${item.label ?? 'secret'}"`).

`renderTableBody(container: HTMLElement, items: DashboardSecretItem[])` — clears `<tbody>` and re-renders rows. Call this on tab change.

**Status tabs:**

Tabs: `['all', 'active', 'viewed', 'expired', 'deleted']` — capitalized labels.

On tab click: update `currentTab`, update `aria-selected` states, filter `allSecrets` and call `renderTableBody`.

Filter logic:
```typescript
function filterSecrets(tab: typeof currentTab): DashboardSecretItem[] {
  if (tab === 'all') return allSecrets;
  return allSecrets.filter((s) => s.status === tab);
}
```

**Step 3 — Add an async `updateDashboardLink()` function inside `createHeader()`, alongside the existing `updateCreateLink()` function:**

```typescript
async function updateDashboardLink(): Promise<void> {
  try {
    const result = await authClient.getSession();
    // Better Auth getSession() returns `any`; use a type guard per CLAUDE.md convention
    const isAuthenticated =
      result !== null &&
      typeof result === 'object' &&
      'data' in result &&
      result.data !== null &&
      typeof result.data === 'object' &&
      'session' in result.data &&
      result.data.session !== null;
    dashboardLink.classList.toggle('hidden', !isAuthenticated);
  } catch {
    dashboardLink.classList.add('hidden');
  }
}
```

**Step 4 — Register the listener and call on init:**

Replace the existing single `routechange` listener registration block with:

```typescript
window.addEventListener('routechange', updateCreateLink);
window.addEventListener('routechange', () => { void updateDashboardLink(); });
updateCreateLink();          // synchronous — runs immediately
void updateDashboardLink();  // async — fires and resolves without blocking render
```

This ensures the Dashboard link is evaluated on every route change (e.g., after login/logout) and on initial page load.
