    Types:
      export type PasswordTier = 'low' | 'medium' | 'high' | 'max';
      export interface PasswordOptions {
        tier: PasswordTier;
        uppercase: boolean;
        numbers: boolean;
        symbols: boolean;
        easyToSay: boolean;   // avoids symbols and numbers; phonetically pronounceable lowercase only
        easyToRead: boolean;  // avoids ambiguous glyphs (1/l/I/|/0/O/B/8/S/5 etc.)
        omitSimilar: boolean; // removes 1/l/I/|/0/O and all visually confusing chars
      }
      export interface GeneratedPassword {
        password: string;
        entropyBits: number;
        bruteForceEstimate: string; // human-readable e.g. "~centuries at 10B guesses/sec"
      }
      export function generatePassword(options: PasswordOptions): GeneratedPassword;

    Tier definitions (length × default charset):
      low    → length 8,  lowercase only (uppercase=false, numbers=false, symbols=false)
      medium → length 12, lowercase+uppercase+numbers (symbols=false)
      high   → length 16, lowercase+uppercase+numbers+symbols
      max    → length 24, lowercase+uppercase+numbers+symbols

    Character set constants (defined as module-level const strings):
      LOWERCASE  = 'abcdefghijklmnopqrstuvwxyz'
      UPPERCASE  = 'ABCDEFGHIJKLMNOPQRSTUVWXYZ'
      NUMBERS    = '0123456789'
      SYMBOLS    = '!@#$%^&*()-_=+[]{}|;:,.<>?'
      PHONETIC   = 'abcdefghjkmnprstuvwyz'  (easy-to-say: omits e,i,l,o,q,x — phonetically ambiguous)
      SIMILAR_RE = /[1lI|0O]/g              (omit-similar filter)
      AMBIGUOUS_RE = /[1lI|0OB8S5]/g       (easy-to-read filter; superset of SIMILAR_RE)

    Charset construction logic (in order):
      1. Start from tier defaults for each char class flag
      2. Apply options.uppercase / options.numbers / options.symbols overrides
         — options.easyToSay forces: uppercase=false, numbers=false, symbols=false, uses PHONETIC for lowercase
         — easyToSay takes precedence over uppercase/numbers/symbols checkboxes
      3. Build raw charset string: concatenate active char class strings (deduplicated)
      4. Apply omitSimilar filter (SIMILAR_RE) if options.omitSimilar === true
      5. Apply easyToRead filter (AMBIGUOUS_RE) if options.easyToRead === true
         (easyToRead is a superset; if both flags set, easyToRead already covers omitSimilar)
      6. Deduplicate final charset (use Set or filter to ensure no duplicate chars)
      7. Guard: if charset.length === 0 after filtering → throw new Error('No characters available with current filter combination')
      8. Compute entropyBits = tier_length * Math.log2(charset.length)  — use EFFECTIVE charset size

    Random char selection — rejection sampling (copy from passphrase.ts pattern):
      function randomChar(charset: string): string {
        const n = charset.length;
        const limit = Math.floor(0xFFFFFFFF / n) * n;
        const buf = new Uint32Array(1);
        let value: number;
        do {
          crypto.getRandomValues(buf);
          value = buf[0]!;
        } while (value >= limit);
        return charset[value % n]!;
      }

    Brute force estimate (10B guesses/sec baseline):
      const GUESSES_PER_SEC = 10_000_000_000;
      const seconds = Math.pow(2, entropyBits) / GUESSES_PER_SEC;
      Thresholds → label:
        < 1        → 'instantly'
        < 60       → '~seconds'
        < 3600     → '~minutes'
        < 86400    → '~hours'
        < 2592000  → '~days'
        < 31536000 → '~months'
        < 3.156e10 → '~decades'
        < 3.156e11 → '~centuries'
        ≥ 3.156e11 → '~eons'
      Display string: `~${label} at 10B guesses/sec`

      5. Uppercase=false on High tier → no uppercase chars in result
